# AI Invoice Recognition & Analysis System

## 📌 Overview
This project is an AI-based system designed to solve the inefficiency of managing paper invoices.  
It integrates OCR recognition, automatic prize checking, AI classification, and spending analysis.

---

## 🚀 Features
- OCR invoice recognition (EasyOCR)
- Automatic prize checking system
- AI-based classification (learning system using learn.csv)
- Data visualization (pie chart, bar chart, line chart)
- Excel report export
- Data management (edit / delete / reset)

---

## 🧠 System Flow
Upload Invoice → OCR Recognition → Classification → Learning → Analysis → Export

---

## 🛠️ Tech Stack
- Python  
- EasyOCR  
- Pandas  
- Matplotlib  
- Tkinter  

---

## 📊 Result
- OCR accuracy: 87% (30 test invoices)
- Supports multiple real-world conditions (clear / wrinkled / low light)
- Provides visualized spending insights

---

## 💡 Project Highlights
- Combines OCR + AI + Data Analysis into one system  
- Personalized learning mechanism (learn.csv)  
- Offline processing (no platform dependency)  
- Practical application for real-life financial tracking
